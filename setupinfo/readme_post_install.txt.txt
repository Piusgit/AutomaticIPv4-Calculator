AutomaticIPv4 Calculator is a powerful tool that helps you with subnetting calculations and Variable Length Subnet Masking (VLSM). It comes with a user-friendly GUI, making it easy for you to perform network design tasks.

Getting Started:
- To launch the application, simply double-click on the "AutomaticIPv4Calculator.exe" file in the installation directory.

Using the App:
- Enter the IP address and CIDR value to calculate subnet information.
- For VLSM calculations, provide the IP address/CIDR, number of subnets, and required hosts per subnet.
- Click the "Calculate" button to perform the calculations.
